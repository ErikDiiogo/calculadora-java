# Calculadora Java

Calculadora de console desenvolvida em Java com as operações:
adição, subtração, multiplicação, divisão, potência e raiz quadrada.

## Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão (com tratamento de divisão por zero)
- Potência
- Raiz quadrada

## Tecnologias
- Java 21
- VS Code
